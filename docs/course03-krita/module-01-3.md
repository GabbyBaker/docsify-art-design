### Using the Tablet with Krita

You may remember from the ["Getting Into Digital Art" learning path](course02-digital-intro/home.md) that a drawing tablet is a tool that you can (and should!) use to make drawing on a computer easier. However, you can't just plug it into your computer and expect it to work. Each application, including Krita, requires a different setup to link the tablet to the application tools.

#### Filter Masks

Krita is primarily a digital painting app, but it has some features that rival Photoshop (which is primarily an image editiing and manipulation app). One of these features is called the **Filter Layer.** 



To practice using this tool, **first** download a royalty-free image from one of these sites:

* [Unsplash](https://unsplash.com)
